## Overview

Instinct API provides AI-powered video editing capabilities that automatically analyze and edit videos by removing unwanted parts like retakes, pauses, and filler content.

**Key Features:**
- AI-driven video analysis and editing
- Automatic removal of retakes, pauses, and filler
- TikTok-style caption generation
- Customizable cutting aggressiveness
- Detailed segment analysis

## Base URL Structure

- **Root:** `/`
- **Health checks:** `/health` or `/api/v1/healthz`
- **Editing:** `/api/v1/editing/...`
