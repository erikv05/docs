## POST /api/v1/editing/generate-shortform-edit

Takes a video URL, analyzes it with AI, removes unwanted parts (retakes, pauses, filler), and returns an edited video.

### Parameters

| Parameter | Type | Required | Description                |
|-----------|------|----------|----------------------------|
| video_url | URL  | âœ… Yes    | Link to the video file     |

### Caption Style Options

When `add_captions` is set to `true`, you can customize the caption appearance:

| Parameter          | Type    | Default | Range   | Description                                      |
|--------------------|---------|---------|---------|--------------------------------------------------|
| font_size          | Number  | 45      | 20-200  | Caption text size                                |
| font_color         | String  | "white" | -       | Text color (hex or name)                         |
| stroke_color       | String  | "black" | -       | Outline color                                    |
| stroke_width       | Number  | 3       | 0-20    | Outline thickness                                |
| position_y_percent | Number  | 0.45    | 0.0-1.0 | Vertical position (0=top, 1=bottom)              |
| max_chars_per_line | Number  | 30      | 10-100  | Text wrapping limit                              |
| uppercase          | Boolean | true    | -       | Convert text to UPPERCASE                        |

### Response

| Field                | Description                                       |
|----------------------|---------------------------------------------------|
| video_url            | Link to your edited video (valid for 24 hours)   |
| original_duration_ms | How long the original video was (milliseconds)    |
| edited_duration_ms   | How long the edited video is (milliseconds)       |
| reduction_percent    | How much was cut (as percentage)                  |
| segments             | List of kept video segments with timing info      |
| analysis             | AI's explanation of what it did                   |

### Segment Information

Each segment in the `segments` array includes:
- Start/end times (in milliseconds and seconds)
- Duration
- Reason for keeping this part

### Example Request

```bash
curl -X POST https://api.instinct.com/api/v1/editing/generate-shortform-edit \
  -H "Content-Type: application/json" \
  -d '{
    "video_url": "https://example.com/video.mp4"
  }'
```

### Notes

- The edited video URL is valid for 24 hours
- Processing time varies based on video length
- Larger `min_pause_duration_ms` values result in less aggressive cutting
- `aggressive_cutting` mode is recommended for fast-paced social media content
