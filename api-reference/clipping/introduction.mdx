## Clipping API

The Clipping API provides a comprehensive suite of tools for AI-powered video processing, from generating highlight reels to searching video content with natural language.

### Key Features

- **Highlight Generation** - Automatically create highlight reels from long-form video content
- **Hook Creation** - Generate engaging social media hooks with cliffhangers
- **Video Search** - Index and search video content using natural language queries
- **Summarization** - Generate AI-powered summaries in multiple formats
- **Transcription** - Extract audio-visual transcripts with timestamps and speaker diarization

### Base URL

```
https://api.usenarrative.ai
```

### API Groups

| Group | Description |
|-------|-------------|
| **Highlights** | Generate highlight reels from long-form content (sports, events) |
| **Hooks** | Create engaging hooks from episodic content (TV series, podcasts) |
| **Analysis** | Summarize videos and extract transcripts |
| **Vision** | Detect ROIs and extract structured metadata from video frames |

### Quick Start

1. **Generate Football Highlights** - Upload a video and get an AI-curated highlight reel
2. **Create Hooks** - Prepare episodes and generate social media hooks
3. **Analyze Videos** - Get summaries or full transcripts
4. **Vision Pipeline** - Detect on-screen elements and extract structured metadata

### Enhanced Highlights Workflow

For maximum accuracy with broadcast video (sports, news, live events), use the Vision pipeline:

```
┌─────────────────┐     ┌──────────────────┐     ┌────────────────────┐
│  Detect ROIs    │ ──▶ │ Extract Metadata │ ──▶ │ Generate Football Highlights│
│  (calibration)  │     │  (play-by-play)  │     │   (with metadata)  │
└─────────────────┘     └──────────────────┘     └────────────────────┘
```

1. **Detect ROIs** - AI identifies scoreboard, clock, and HUD elements
2. **Extract Metadata** - Monitor video and generate timestamped events
3. **Generate Football Highlights** - Use metadata for precise, context-aware clips
